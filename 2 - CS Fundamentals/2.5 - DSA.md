## 🧱 Data Structures

### Array & Slice

```
Access:  O(1)   — direct index
Search:  O(n)   — linear scan
         O(log n) — binary search (sorted)
Insert:  O(n)   — shift elements
Delete:  O(n)   — shift elements
```

**Khi dùng**: Random access, fixed size, cache-friendly iteration.

### Hash Map / Hash Set

```
Insert:  O(1) avg, O(n) worst (collision)
Delete:  O(1) avg
Lookup:  O(1) avg
```

**Pattern**: Frequency count, seen-before check, complement lookup (Two Sum).

```go
freq := make(map[byte]int)
for _, c := range s {
    freq[c]++
}
```

### Stack

```
Push:  O(1)
Pop:   O(1)
Peek:  O(1)
```

**Khi dùng**: Matching brackets, undo, DFS iterative, monotonic problems.

**Monotonic Stack**: Maintain elements in increasing/decreasing order → tìm next greater/smaller.

```go
stack := []int{}
for i, n := range nums {
    for len(stack) > 0 && nums[stack[len(stack)-1]] < n {
        idx := stack[len(stack)-1]
        stack = stack[:len(stack)-1]
        // nums[idx] < n → n is next greater for idx
    }
    stack = append(stack, i)
}
```

### Heap / Priority Queue

```
Insert:     O(log n)
Delete min: O(log n)
Peek min:   O(1)
Heapify:    O(n)
```

**Pattern Top-K**: Min-heap size K → O(n log k) thay vì O(n log n sort).

```go
import "container/heap"

// Min-heap: negate values cho max-heap
type MinHeap []int
func (h MinHeap) Len() int           { return len(h) }
func (h MinHeap) Less(i, j int) bool { return h[i] < h[j] }
func (h MinHeap) Swap(i, j int)      { h[i], h[j] = h[j], h[i] }
func (h *MinHeap) Push(x any)        { *h = append(*h, x.(int)) }
func (h *MinHeap) Pop() any {
    old := *h; n := len(old)
    x := old[n-1]; *h = old[:n-1]; return x
}
```

### Linked List

```
Access:  O(n)
Search:  O(n)
Insert:  O(1)  — with pointer
Delete:  O(1)  — with pointer
```

**Patterns**:
- **Fast/Slow pointer**: Cycle detection, find middle
- **Dummy head**: Simplify edge cases
- **Reverse**: Iterative với prev/curr/next

```go
// Reverse linked list — iterative
var prev *ListNode
curr := head
for curr != nil {
    next := curr.Next
    curr.Next = prev
    prev = curr
    curr = next
}
return prev
```

### Tree

```
Search (BST):  O(log n) avg, O(n) worst
Insert (BST):  O(log n) avg
DFS:           O(n) time, O(h) space
BFS:           O(n) time, O(w) space (w = max width)
```

**DFS Traversals**:
- Preorder (root, left, right): serialize, copy tree
- Inorder (left, root, right): BST → sorted sequence
- Postorder (left, right, root): delete tree, calculate subtree

```go
// DFS template — most tree problems
func dfs(node *TreeNode) int {
    if node == nil {
        return 0  // base case
    }
    left := dfs(node.Left)
    right := dfs(node.Right)
    // process current node
    return max(left, right) + 1
}
```

### Trie (Prefix Tree)

```
Insert: O(n)   — n = word length
Search: O(n)
Space:  O(total characters)
```

**Khi dùng**: Autocomplete, prefix search, word dictionary.

```go
type TrieNode struct {
    children [26]*TrieNode
    isEnd    bool
}
type Trie struct{ root *TrieNode }

func (t *Trie) Insert(word string) {
    node := t.root
    for _, c := range word {
        i := c - 'a'
        if node.children[i] == nil {
            node.children[i] = &TrieNode{}
        }
        node = node.children[i]
    }
    node.isEnd = true
}
```

### Graph

```
DFS/BFS:   O(V + E)
Dijkstra:  O((V + E) log V)  — with min-heap
Bellman-Ford: O(V × E)       — handles negative edges
Union-Find: O(α(n)) ≈ O(1)  — amortized
```

**Representations**:
- **Adjacency list**: `map[int][]int` — sparse graphs
- **Adjacency matrix**: `[][]int` — dense graphs, O(1) edge check

## 🔄 Graph Algorithms

### BFS — Shortest Path (Unweighted)

```go
q := []int{start}
visited := map[int]bool{start: true}
steps := 0
for len(q) > 0 {
    for size := len(q); size > 0; size-- {
        node := q[0]; q = q[1:]
        if node == target { return steps }
        for _, next := range graph[node] {
            if !visited[next] {
                visited[next] = true
                q = append(q, next)
            }
        }
    }
    steps++
}
```

### Topological Sort (Kahn's BFS)

```go
// Detect cycle + order for DAG
inDegree := make([]int, n)
for _, e := range edges {
    inDegree[e[1]]++
}
q := []int{}
for i, d := range inDegree {
    if d == 0 { q = append(q, i) }
}
order := []int{}
for len(q) > 0 {
    node := q[0]; q = q[1:]
    order = append(order, node)
    for _, next := range graph[node] {
        inDegree[next]--
        if inDegree[next] == 0 { q = append(q, next) }
    }
}
// len(order) == n → no cycle
```

### Union-Find (Disjoint Set)

```go
type UnionFind struct{ parent, rank []int }

func NewUF(n int) *UnionFind {
    p := make([]int, n)
    for i := range p { p[i] = i }
    return &UnionFind{parent: p, rank: make([]int, n)}
}

func (uf *UnionFind) Find(x int) int {
    if uf.parent[x] != x {
        uf.parent[x] = uf.Find(uf.parent[x]) // path compression
    }
    return uf.parent[x]
}

func (uf *UnionFind) Union(x, y int) bool {
    px, py := uf.Find(x), uf.Find(y)
    if px == py { return false } // already same component
    if uf.rank[px] < uf.rank[py] { px, py = py, px }
    uf.parent[py] = px
    if uf.rank[px] == uf.rank[py] { uf.rank[px]++ }
    return true
}
```

### Dijkstra (Weighted Shortest Path)

```go
dist := make([]int, n)
for i := range dist { dist[i] = math.MaxInt }
dist[src] = 0

pq := &MinHeap{{0, src}}
heap.Init(pq)

for pq.Len() > 0 {
    item := heap.Pop(pq).([2]int)
    d, node := item[0], item[1]
    if d > dist[node] { continue } // stale entry
    for _, e := range graph[node] {
        next, w := e[0], e[1]
        if dist[node]+w < dist[next] {
            dist[next] = dist[node] + w
            heap.Push(pq, [2]int{dist[next], next})
        }
    }
}
```

## ↩️ Backtracking

### Template

```go
func backtrack(start int, current []int) {
    // Base case
    if condition(current) {
        result = append(result, append([]int{}, current...))
        return
    }

    for i := start; i < n; i++ {
        // Pruning
        if shouldSkip(i, current) { continue }

        // Choose
        current = append(current, nums[i])

        // Explore
        backtrack(i+1, current)  // i+1: no reuse, i: can reuse

        // Unchoose (backtrack)
        current = current[:len(current)-1]
    }
}
```

### Dedup trong Backtracking

```go
// Khi input có duplicates và không muốn duplicate results
sort.Ints(nums)
for i := start; i < n; i++ {
    if i > start && nums[i] == nums[i-1] { continue } // skip duplicate
    // ...
}
```

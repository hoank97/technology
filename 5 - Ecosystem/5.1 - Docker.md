## Image vs Container

| | Image | Container |
|---|---|---|
| Bản chất | Template read-only | Running instance của image |
| Trạng thái | Không thay đổi | Có thể thay đổi (write layer) |
| Tạo bằng | `docker build` | `docker run` |

## Dockerfile

```dockerfile
FROM golang:1.22-alpine AS builder
WORKDIR /app
COPY go.mod go.sum ./
RUN go mod download
COPY . .
RUN CGO_ENABLED=0 go build -o app .

FROM alpine:latest
WORKDIR /app
COPY --from=builder /app/app .
EXPOSE 8080
CMD ["./app"]
```

### Key instructions

| | |
|---|---|
| `FROM` | Base image |
| `COPY` | Copy files |
| `RUN` | Run commands |
| `EXPOSE` | Document port |
| `CMD` | Default command |

## Docker Compose

```yaml
version: "3.8"
services:
  app:
    build: .
    ports:
      - "8080:8080"
    depends_on:
      - db
      - redis
  db:
    image: postgres:15
    environment:
      POSTGRES_DB: myapp
      POSTGRES_PASSWORD: secret
  redis:
    image: redis:7
```

```bash
docker compose up -d      # Start all services
docker compose down       # Stop all services
docker compose logs -f    # Follow logs
```

## Common Commands

```bash
docker build -t myapp .           # Build image
docker run -p 8080:8080 myapp    # Run container
docker ps                         # List running containers
docker images                      # List images
docker compose up -d               # Start compose
docker compose down                # Stop compose
docker exec -it container sh       # Shell vào container
docker logs -f container           # Follow logs
docker network ls                  # List networks
```

# 6.4 Go Embed

## Overview

`//go:embed` cho phép embed files/directories vào binary tại **compile time** — không cần runtime file reading.

> Phù hợp: static assets (HTML, CSS, templates, certificates, version files).

---

## Syntax

```go
import _ "embed"

//go:embed version.txt
var version string

//go:embed config.yaml
var configData []byte

//go:embed static
var staticFiles embed.FS
```

---

## Embedding Patterns

### Single file → string

```go
//go:embed version.txt
var version string

fmt.Println("App version:", version)
// version.txt content compiled into binary
```

### Single file → []byte

```go
//go:embed cert/server.crt
var certData []byte

// Dùng với TLS
cert, err := tls.X509KeyPair(certData, keyData)
```

### Multiple files (glob)

```go
//go:embed data/*.json
var dataFiles embed.FS

// Đọc từng file
content, err := dataFiles.ReadFile("data/config.json")
```

### Entire directory

```go
//go:embed public
var publicFiles embed.FS

// List directory
entries, err := publicFiles.ReadDir("public")
for _, e := range entries {
    fmt.Println(e.Name(), e.IsDir())
}
```

---

## embed.FS API

```go
import "embed"

var files embed.FS

// Read single file
data, err := files.ReadFile("static/index.html")

// Read directory contents
entries, err := files.ReadDir("static")
for _, e := range entries {
    info, _ := e.Info()
    fmt.Println(e.Name(), e.IsDir(), info.Size())
}

// Match glob pattern
matches, err := files.Glob("templates/*.tmpl")
```

## HTTP Static Server

### Basic

```go
import (
    "embed"
    "net/http"
)

//go:embed public
var static embed.FS

func main() {
    fs := http.FS(static)
    http.Handle("/", http.FileServer(fs))
    http.ListenAndServe(":8080", nil)
}
```

### SPA (Single Page App — fallback to index.html)

```go
func spaHandler(fs embed.FS) http.Handler {
    return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        path := r.URL.Path

        // Try exact file
        if _, err := fs.Open(path); err == nil {
            http.FileServer(fs).ServeHTTP(w, r)
            return
        }

        // Fallback to index.html for SPA routes
        path = "public/index.html"
        f, err := fs.Open(path)
        if err != nil {
            http.NotFound(w, r)
            return
        }
        defer f.Close()
        http.ServeContent(w, r, "index.html", 0, f)
    })
}
```

## Templates

```go
//go:embed templates/*
var templateFS embed.FS

func main() {
    tmpl, err := template.ParseFS(templateFS, "templates/*.html")
    if err != nil {
        log.Fatal(err)
    }

    tmpl.ExecuteTemplate(w, "home.html", data)
}
```

## Certificates (TLS)

```go
//go:embed cert/server.crt
var certFile embed.FS

//go:embed cert/server.key
var keyFile embed.FS

func loadCerts() (tls.Certificate, error) {
    cert, err := certFile.ReadFile("cert/server.crt")
    if err != nil {
        return tls.Certificate{}, err
    }
    key, err := keyFile.ReadFile("cert/server.key")
    if err != nil {
        return tls.Certificate{}, err
    }
    return tls.X509KeyPair(cert, key)
}
```

## Configuration

```go
//go:embed config/*.yaml
var configFS embed.FS

func loadConfig(name string) (*Config, error) {
    data, err := configFS.ReadFile("config/" + name + ".yaml")
    if err != nil {
        return nil, err
    }
    return parseConfig(data)
}
```

## Comparison: Embed vs Runtime Read

| | Embed | Runtime Read |
|---|-------|--------------|
| Timing | Compile time | Runtime |
| File on disk | ❌ Not needed | ✅ Needed |
| Binary size | ✅ Larger (assets included) | ✅ Smaller |
| Update assets | ❌ Must recompile | ✅ Edit file directly |
| Security | ✅ Embedded = immutable | ⚠️ File can be modified |
| Portability | ✅ Single binary | ❌ Need all asset files |
| Build time | ⚠️ Slower (compile assets) | ✅ Faster |

### Khi nào dùng?

| Use case | Embed? |
|----------|--------|
| Certificates / keys | ✅ Yes (security) |
| Version / build info | ✅ Yes (always available) |
| Static HTML/CSS/JS | ⚠️ Depends (update frequency) |
| Large media files | ❌ No (bloat binary) |
| Config files | ⚠️ Depends (update frequency) |

## Constraints

### Ignore warnings

```bash
# Nếu file không tìm thấy → compile error
# "no required module provides package ..."

# Must match exact path relative to .go file
```

### .gitignore

```go
//go:embed .env (❌ ERROR — .env thường gitignored)
//go:embed config/  ✅ Fine
```

## 📌 Tóm tắt

```
go:embed
│
├── Patterns
│   ├── //go:embed file.txt → string variable
│   ├── //go:embed file.bin → []byte variable
│   ├── //go:embed dir/* → embed.FS
│   └── //go:embed dir → embed.FS (all contents)
│
├── embed.FS API
│   ├── ReadFile(path) → []byte
│   ├── ReadDir(path) → []DirEntry
│   └── Glob(pattern) → []string
│
├── Common Uses
│   ├── HTTP static server: http.FS(fs)
│   ├── Templates: template.ParseFS()
│   ├── Certificates: tls.X509KeyPair(data, key)
│   └── Config files: yaml.Unmarshal(data)
│
└── Trade-offs
    ├── ✅ Single binary, immutable, no missing files
    ├── ✅ Security (certs embedded)
    └── ⚠️ Larger binary, must recompile to update
```

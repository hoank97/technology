## Embedding Patterns

### Single file → string

```go
//go:embed version.txt
var version string

fmt.Println("App version:", version)
// version.txt content compiled into binary
```

### Single file → []byte

```go
//go:embed cert/server.crt
var certData []byte

// Dùng với TLS
cert, err := tls.X509KeyPair(certData, keyData)
```

### Multiple files (glob)

```go
//go:embed data/*.json
var dataFiles embed.FS

// Đọc từng file
content, err := dataFiles.ReadFile("data/config.json")
```

### Entire directory

```go
//go:embed public
var publicFiles embed.FS

// List directory
entries, err := publicFiles.ReadDir("public")
for _, e := range entries {
    fmt.Println(e.Name(), e.IsDir())
}
```

## HTTP Static Server

### Basic

```go
import (
    "embed"
    "net/http"
)

//go:embed public
var static embed.FS

func main() {
    fs := http.FS(static)
    http.Handle("/", http.FileServer(fs))
    http.ListenAndServe(":8080", nil)
}
```

### SPA (Single Page App — fallback to index.html)

```go
func spaHandler(fs embed.FS) http.Handler {
    return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        path := r.URL.Path

        // Try exact file
        if _, err := fs.Open(path); err == nil {
            http.FileServer(fs).ServeHTTP(w, r)
            return
        }

        // Fallback to index.html for SPA routes
        path = "public/index.html"
        f, err := fs.Open(path)
        if err != nil {
            http.NotFound(w, r)
            return
        }
        defer f.Close()
        http.ServeContent(w, r, "index.html", 0, f)
    })
}
```

## Certificates (TLS)

```go
//go:embed cert/server.crt
var certFile embed.FS

//go:embed cert/server.key
var keyFile embed.FS

func loadCerts() (tls.Certificate, error) {
    cert, err := certFile.ReadFile("cert/server.crt")
    if err != nil {
        return tls.Certificate{}, err
    }
    key, err := keyFile.ReadFile("cert/server.key")
    if err != nil {
        return tls.Certificate{}, err
    }
    return tls.X509KeyPair(cert, key)
}
```

## Comparison: Embed vs Runtime Read

| | Embed | Runtime Read |
|---|-------|--------------|
| Timing | Compile time | Runtime |
| File on disk | ❌ Not needed | ✅ Needed |
| Binary size | ✅ Larger (assets included) | ✅ Smaller |
| Update assets | ❌ Must recompile | ✅ Edit file directly |
| Security | ✅ Embedded = immutable | ⚠️ File can be modified |
| Portability | ✅ Single binary | ❌ Need all asset files |
| Build time | ⚠️ Slower (compile assets) | ✅ Faster |

### Khi nào dùng?

| Use case | Embed? |
|----------|--------|
| Certificates / keys | ✅ Yes (security) |
| Version / build info | ✅ Yes (always available) |
| Static HTML/CSS/JS | ⚠️ Depends (update frequency) |
| Large media files | ❌ No (bloat binary) |
| Config files | ⚠️ Depends (update frequency) |

## Branching

```bash
git branch                  # List branches
git branch feature/login   # Create branch
git checkout feature/login # Switch branch
git switch feature/login   # Switch (new syntax)

# Shortcut
git checkout -b feature/login  # Create + switch
git switch -c feature/login   # Create + switch (new)
```

## Merge vs Rebase

### Merge

```bash
git checkout main
git merge feature/login
```

```
main:    A---B---C
                 \
feature:         D---E
                        \
merged:           A---B---C---M
```

→ **Preserve history**, tạo merge commit

### Rebase

```bash
git checkout feature/login
git rebase main
```

```
main:    A---B---C
feature:         D'---E'
```

→ **Linear history**, đẹp hơn nhưng **rewrites history**

⚠️ **Không rebase** commits đã push lên remote!

## Undo Changes

```bash
git checkout -- file      # Discard local changes
git restore file          # Discard local changes (new syntax)
git reset HEAD file       # Unstage file
git reset --soft HEAD~1   # Undo commit, keep changes staged
git reset --mixed HEAD~1   # Undo commit, keep changes unstaged
git reset --hard HEAD~1   # Undo commit + changes (⚠️ DESTRUCTIVE)
git revert HEAD           # Create new commit that undoes previous
```

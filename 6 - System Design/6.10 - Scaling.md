## Locking vs Optimistic vs Pessimistic

| | Optimistic | Pessimistic |
|---|---|---|
| Strategy | Detect conflict after | Prevent conflict before |
| Lock | Không lock | Lock resource |
| Error | Return conflict error | Block until lock released |
| Use case | Low contention | High contention |
| Example | `UPDATE ... WHERE version = v` | `SELECT FOR UPDATE` |

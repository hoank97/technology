## 1. Immediate Retry

Gửi lại request **ngay lập tức** sau failure.

```go
err := callAPI()
if err != nil {
    err = callAPI() // retry ngay
}
```

### Best for

- Very brief network glitches
- Xảy ra rất hiếm

### Trade-off

- ⚠️ **Potentially crash the server** vì request overload

---

## 2. Fixed Delay

Đợi **một khoảng thời gian cố định** giữa mỗi attempt.

```go
for attempt := 0; attempt < maxRetries; attempt++ {
    err := callAPI()
    if err == nil {
        return nil
    }
    if attempt < maxRetries-1 {
        time.Sleep(1 * time.Second) // Đợi 1s giữa mỗi lần
    }
}
```

### Best for

- Services where recovery time is predictable
- Khi biết thời gian recovery

---

## 3. Exponential Backoff

Thời gian đợi **tăng theo exponential** với mỗi failure.

```
Attempt 1: wait = 100ms
Attempt 2: wait = 200ms
Attempt 3: wait = 400ms
Attempt 4: wait = 800ms
...
```

```go
for attempt := 0; attempt < maxRetries; attempt++ {
    err := callAPI()
    if err == nil {
        return nil
    }

    if attempt < maxRetries-1 {
        delay := time.Duration(math.Pow(2, float64(attempt))) * 100 * time.Millisecond
        time.Sleep(delay)
    }
}
```

### Best for

- Avoid thundering herd
- Reduce stress in the target server
- Most scenarios

---

## 4. Exponential Backoff with Jitter

Thêm **random "noise"** vào delay để **completely avoid thundering herd**.

```
Without Jitter: 1000 clients all retry at 100ms, 200ms, 400ms...
With Jitter:    Random spread → smooth load
```

```go
func backoffWithJitter(attempt int) time.Duration {
    base := time.Duration(math.Pow(2, float64(attempt))) * 100 * time.Millisecond
    // Random: 0% - 100% của base delay
    jitter := time.Duration(rand.Float64() * float64(base))
    return base + jitter
}

for attempt := 0; attempt < maxRetries; attempt++ {
    err := callAPI()
    if err == nil {
        return nil
    }
    if attempt < maxRetries-1 {
        time.Sleep(backoffWithJitter(attempt))
    }
}
```

### Jitter Types

| Type | Formula | Use case |
|------|---------|----------|
| **Full jitter** | `random(0, base)` | Best spread |
| **Equal jitter** | `base/2 + random(0, base/2)` | Conservative |
| **Decorrelated** | `random(base, previous*3)` | Very aggressive |

## Retry with Context

```go
func callWithRetry(ctx context.Context, maxRetries int) error {
    for attempt := 0; attempt < maxRetries; attempt++ {
        select {
        case <-ctx.Done():
            return ctx.Err()
        default:
        }

        err := callAPI()
        if err == nil {
            return nil
        }

        if attempt < maxRetries-1 {
            delay := backoffWithJitter(attempt)
            select {
            case <-ctx.Done():
                return ctx.Err()
            case <-time.After(delay):
            }
        }
    }
    return fmt.Errorf("after %d retries", maxRetries)
}
```

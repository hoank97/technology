## Pool Settings

| Setting | Mô tả |
|---------|--------|
| **MaxOpenConns** | Tổng số connections tối đa |
| **MaxIdleConns** | Số connections giữ idle |
| **ConnMaxLifetime** | Recycle sau X time |
| **ConnMaxIdleTime** | Close idle connection sau X time |

### Sizing pool

```
Formula:
pool_size = (number_of_cores × 2) + effective_spindle_count

Ví dụ: 4 cores + SSD (no spindle limit)
→ pool_size = (4 × 2) + 0 = 8
```

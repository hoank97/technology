## TCP Socket

**Connection-oriented** — phải establish connection trước khi communicate.

```
Client                          Server
  │                               │
  │  ──── SYN ─────────────────→  │  (connect)
  │  ←─── SYN-ACK ─────────────  │
  │  ──── ACK ─────────────────→  │  (connection established)
  │                               │
  │  ════ Data Transfer ═══════════│
  │                               │
  │  ──── FIN ─────────────────→  │  (close)
  │  ←─── FIN-ACK ─────────────  │
  │  ──── ACK ─────────────────→  │
```

### Đặc điểm

| | |
|---|---|
| **Reliable** | Data không mất, không trùng |
| **Ordered** | Data đến đúng thứ tự |
| **Connection** | Phải connect trước |
| **Use case** | HTTP, database, file transfer |

### Go example

```go
// Server
ln, _ := net.Listen("tcp", ":8080")
conn, _ := ln.Accept()
buf := make([]byte, 1024)
conn.Read(buf)
conn.Write([]byte("response"))
conn.Close()

// Client
conn, _ := net.Dial("tcp", "localhost:8080")
conn.Write([]byte("hello"))
```

## UDP Socket

**Connectionless** — không cần establish connection.

### Đặc điểm

| | |
|---|---|
| **Unreliable** | Data có thể mất, trùng |
| **No connection** | Gửi thẳng, không handshaking |
| **Faster** | Không overhead của connection |
| **Use case** | DNS, video streaming, VoIP |

```go
// Server (UDP)
addr, _ := net.ResolveUDPAddr("udp", ":8080")
conn, _ := net.ListenUDP("udp", addr)
buf := make([]byte, 1024)
conn.ReadFromUDP(buf)

// Client (UDP)
conn, _ := net.Dial("udp", "localhost:8080")
conn.Write([]byte("hello"))
```

## TCP vs UDP

| | TCP | UDP |
|---|---|---|
| Connection | Connection-oriented | Connectionless |
| Reliability | ✅ Reliable | ❌ Unreliable |
| Ordering | ✅ Ordered | ❌ No order |
| Speed | Chậm hơn | Nhanh hơn |
| Overhead | Connection setup | Không |
| Use case | HTTP, email | DNS, VoIP, gaming |

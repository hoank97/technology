## Container

- Chia sẻ **kernel** của host OS
- Nhẹ hơn VM nhiều
- Container runtime: **Docker**, **containerd**

```
┌──────────────────────────────────────┐
│              Host OS                 │
│            (Linux Kernel)            │
├─────────────────┬───────────────────┤
│    Container 1  │    Container 2    │
│  ┌───────────┐  │  ┌───────────┐   │
│  │    App    │  │  │    App    │   │
│  ├───────────┤  │  ├───────────┤   │
│  │  Binaries │  │  │  Binaries │   │
│  ├───────────┤  │  ├───────────┤   │
│  │   libs    │  │  │   libs    │   │
│  └───────────┘  │  └───────────┘   │
└─────────────────┴───────────────────┘
```

---

## So sánh

| | VM | Container |
|---|---|---|
| **Isolation** | Full OS, cao | Process-level |
| **Size** | GB (full OS) | MB (chỉ app + libs) |
| **Start time** | Phút | Giây |
| **Performance** | Overhead của hypervisor | Gần native |
| **Resource usage** | Cao | Thấp |
| **Boot** | Full boot | Instant |

---

## Khi nào dùng gì?

| | VM | Container |
|---|---|---|
| Legacy app (full OS) | ✅ | ❌ |
| Microservices | ❌ | ✅ |
| Dev/Testing | ❌ | ✅ |
| Strong isolation | ✅ | ❌ (cùng kernel) |
| Run multiple OS | ✅ | ❌ |

## TCP vs UDP

| | TCP | UDP |
|---|---|---|
| Connection | Connection-oriented | Connectionless |
| Reliability | ✅ Reliable | ❌ Unreliable |
| Ordering | ✅ Ordered | ❌ No order |
| Speed | Chậm hơn | Nhanh hơn |
| Overhead | Connection setup | Không |
| Use case | HTTP, email | DNS, VoIP, gaming |

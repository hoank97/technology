## Protocol Buffers

### .proto file

```protobuf
syntax = "proto3";

service UserService {
    rpc GetUser(GetUserRequest) returns (UserResponse);
    rpc CreateUser(CreateUserRequest) returns (UserResponse);
    rpc StreamUsers(StreamRequest) returns (stream UserResponse);  // Server streaming
}

message GetUserRequest {
    string id = 1;
}

message UserResponse {
    string id = 1;
    string name = 2;
    string email = 3;
}
```

### Generated code

```bash
protoc --go_out=. --go-grpc_out=. user.proto
# → user.pb.go (serialization)
# → user_grpc.go (client/server stubs)
```

## gRPC vs REST

| | REST | gRPC |
|---|---|---|
| Protocol | HTTP/1.1 hoặc HTTP/2 | HTTP/2 (bắt buộc) |
| Format | JSON / XML | Protocol Buffers (binary) |
| Schema | Optional (OpenAPI) | Required (.proto) |
| Code gen | External tools | Built-in (protoc) |
| Streaming | Server-Sent Events | Native bi-directional |
| Browser support | ✅ Universal | ❌ Cần grpc-web proxy |
| Human readable | ✅ JSON | ❌ Binary |

## Streaming

```protobuf
// Client streaming
rpc CreateUsers(stream CreateUserRequest) returns (CreateUsersResponse);

// Server streaming
rpc GetUpdates(Empty) returns (stream Update);

// Bi-directional streaming
rpc Chat(stream ChatMessage) returns (stream ChatMessage);
```

## Error Handling

```go
// Client
resp, err := client.GetUser(ctx, &pb.GetUserRequest{Id: "123"})
if err != nil {
    statusErr, ok := status.FromError(err)
    if statusErr.Code() == codes.NotFound {
        return nil, fmt.Errorf("user not found")
    }
    return nil, err
}
```

```go
// Server
if user == nil {
    return nil, status.Errorf(codes.NotFound, "user %s not found", req.Id)
}
```

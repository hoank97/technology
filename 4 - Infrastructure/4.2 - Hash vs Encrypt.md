## Encrypt

**Two-way function** — **có thể** decrypt để lấy lại original data.

### Đặc điểm

| | |
|---|---|
| **Reversible** | ✅ Có |
| **Key needed** | ✅ Có (salt/key) |
| **Output length** | Không cố định |
| **Use case** | Protecting sensitive data |

### Ví dụ

- **AES**: symmetric encryption (fast, bulk data)
- **RSA**: asymmetric (public/private key)

```go
// AES example
key := []byte("32-byte-secret-key-here!!!!")
plaintext := []byte("hello")

block, _ := aes.NewCipher(key)
stream := cipher.NewCFBEncrypter(block, iv)
ciphertext := make([]byte, len(plaintext))
stream.XORKeyStream(ciphertext, plaintext)
```

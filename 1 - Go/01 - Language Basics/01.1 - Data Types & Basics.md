# 01.1 Data Types & Basics



- **Dynamic size** — kích thước có thể thay đổi
- **Là một struct**:

```go
type slice struct {
    ptr *T   // pointer tới underlying array
    len int  // số phần tử đang dùng
    cap int  // sức chứa tối đa
}
```

- `len`: số phần tử đang dùng
- `cap`: sức chứa tối đa trước khi realloc
- **Reference semantics** — points to an underlying array

### Cheap to pass around

```go
func g(s []int) {
    s[0] = 100
}

func main() {
    s := []int{1, 2, 3}
    g(s)
    fmt.Println(s) // [100 2 3] — caller thấy thay đổi
}
```

### ⚠️ Risk

- Nếu function modify elements → caller thấy thay đổi
- Nếu `append` triggers reallocation → local pointer thay đổi → **caller header vẫn trỏ old array**

## Nil vs Empty

| Type | Ý nghĩa |
|------|---------|
| `nil slice` (`var s []int`) | Chưa init, không có underlying data |
| `empty slice` (`s := []int{}`) | Đã init, nhưng chứa 0 phần tử |

---

## Value Type vs Reference Type

| | Value Type | Reference Type |
|---|---|---|
| Copy | Copy toàn bộ dữ liệu | Copy pointer → shared underlying data |
| Khi truyền | Tạo bản copy mới | Chia sẻ data gốc |
| Ví dụ | Array, int, struct | Slice, map, channel, pointer |

---

## Struct Embedding vs Inheritance

Go **không có inheritance**. Thay vào đó dùng:

### Struct Embedding

```go
type Animal struct {
    Name string
}

func (a Animal) Speak() {
    fmt.Println("Animal speaks")
}

type Dog struct {
    Animal // embedded struct — như "extends"
}
```

- Gọi `Dog{Name: "Buddy"}.Speak()` → `Animal.Speak()`
- **Shadowing**: nếu Dog có field trùng tên với Animal → dùng Dog's field

### So sánh

| | Inheritance (Java/C++) | Struct Embedding (Go) |
|---|---|---|
| Syntax | `class Dog extends Animal` | `Dog Animal` |
| Multiple | Không | Có (multiple embedding) |
| Method override | Override keyword | Shadowing (không polymorphism) |

## Interface{}

- Định nghĩa tập method
- `interface{}` = `any` (Go 1.18+)
- **Mọi type** đều implement `interface{}`

```go
var x interface{} = "hello"  // string implements interface{}
```

---

## Type Assertion vs Type Switch

### Type Assertion — check 1 type cụ thể

```go
func handle(x any) {
    s, ok := x.(string)
    if !ok {
        fmt.Println("not a string")
        return
    }
    fmt.Println("string:", s)
}
```

### Type Switch — check nhiều type

```go
switch v := x.(type) {
case int:
    fmt.Println("int:", v)
case string:
    fmt.Println("string:", v)
default:
    fmt.Println("unknown type")
}
```

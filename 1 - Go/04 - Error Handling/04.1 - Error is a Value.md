## Check Error

```go
// ❌ Bad: Ignore error
data, _ := json.Marshal(v)

// ✅ Good
data, err := json.Marshal(v)
if err != nil {
    return fmt.Errorf("marshal: %w", err)
}

// ✅ Check với comparison
if errors.Is(err, os.ErrNotExist) {
    // handle not found
}
```

---

## Handle vs Ignore

| | Nên? | Lý do |
|---|---|---|
| Handle + return | ✅ | Giữ error chain |
| Log + continue | ✅ | Khi có fallback |
| Ignore với `_` | ❌ | Mất thông tin lỗi |

---

## Best Practices

1. **Return error** — don't panic for expected failures
2. **Wrap error** với `%w` để giữ chain
3. **Check error == nil** trước khi dùng
4. **Log + return** — đừng chỉ log

# 06.5 HTTP Server & Client

## net/http Overview

`net/http` là HTTP implementation chính thức của Go — từ HTTP/1.1 server/client đến handler patterns.

### HTTP Handler model

```go
// http.Handler — interface cốt lõi
type Handler interface {
    ServeHTTP(ResponseWriter, *Request)
}

// http.HandlerFunc — adapter: biến function thành Handler
type HandlerFunc func(ResponseWriter, *Request)

func (f HandlerFunc) ServeHTTP(w ResponseWriter, r *Request) {
    f(w, r) // delegate to function
}
```

> Mọi thứ trong `net/http` xoay quanh `Handler` interface. Server xử lý request bằng cách gọi `handler.ServeHTTP(w, r)`.

## Server — Setup & Lifecycle

### Basic HTTP server

```go
http.HandleFunc("/hello", func(w http.ResponseWriter, r *http.Request) {
    name := r.URL.Query().Get("name")
    fmt.Fprintf(w, "Hello %s", name)
})

http.ListenAndServe(":8080", nil)
// ⚠️ ListenAndServe nhận handler cuối cùng nil → dùng DefaultServeMux
```

### Server with custom configuration

```go
srv := &http.Server{
    Addr:         ":8080",
    Handler:      myHandler,
    ReadTimeout:  5 * time.Second,
    WriteTimeout: 10 * time.Second,
    IdleTimeout:  120 * time.Second,
    MaxHeaderBytes: 1 << 20, // 1MB
}

if err := srv.ListenAndServe(); err != nil {
    if err != http.ErrServerClosed {
        log.Fatal(err)
    }
}
```

### Graceful Shutdown — không drop requests

```go
srv := &http.Server{Addr: ":8080", Handler: handler}

// Tạo context với timeout cho shutdown
ctx, stop := signal.NotifyContext(context.Background(), syscall.SIGINT, syscall.SIGTERM)
defer stop()

// Graceful shutdown: stop accepting new, đợi in-flight hoàn thành
go func() {
    if err := srv.ListenAndServe(); err != nil && err != http.ErrServerClosed {
        log.Fatalf("server error: %v", err)
    }
}()

<-ctx.Done() // nhận SIGINT/SIGTERM
stop()

// Shutdown với context deadline
shutdownCtx, cancel := context.WithTimeout(context.Background(), 30*time.Second)
defer cancel()

if err := srv.Shutdown(shutdownCtx); err != nil {
    log.Fatalf("shutdown error: %v", err)
}
log.Println("server gracefully stopped")
```

### Server behavior — internal details

```
Request lifecycle:
  1. Accept TCP connection
  2. Spawn goroutine (per connection, pre-Go 1.21; now per-request)
  3. Read & parse HTTP request line + headers
  4. Find handler via ServeMux (routing)
  5. Call handler.ServeHTTP(w, r)
  6. Handler writes response
  7. Goroutine returns (connection reused if keep-alive, else closed)
```

## HTTP Request — Parsing

### Path parameters & Query strings

```go
// Query string: /search?q=golang&lang=en
r.URL.Query().Get("q")       // "golang"
r.URL.Query().Get("lang")    // "en"
r.URL.Query().Has("lang")    // true

// ⚠️ Query() parse mỗi lần gọi — cache nếu gọi nhiều lần
values := r.URL.Query()      // url.Values (map[string][]string)

// Form values (POST application/x-www-form-urlencoded hoặc GET query)
r.ParseForm()
r.FormValue("email")         // merged GET + POST values
r.PostFormValue("email")     // POST only

// Multipart form (file upload)
r.ParseMultipartForm(32 << 20) // max 32MB
r.MultipartForm.File["avatar"]
r.MultipartForm.Value["username"]
```

### Request headers

```go
r.Header.Get("Content-Type")    // "application/json"
r.Header.Get("Authorization")
r.Header.Get("Accept-Encoding")

// Common headers shortcuts
r.Host      // "example.com:8080"
r.Method    // "GET", "POST", "PUT"
r.RemoteAddr // "192.168.1.1:12345"
r.UserAgent()
r.Referer()
```

### Body reading

```go
// Read entire body
body, err := io.ReadAll(r.Body)

// Read with size limit (prevent large body DoS)
body, err := io.ReadAll(io.LimitReader(r.Body, 1<<20)) // max 1MB

// Decode JSON directly from body
var req Request
dec := json.NewDecoder(r.Body)
dec.DisallowUnknownFields()
err := dec.Decode(&req)

// Close body (luôn luôn)
defer r.Body.Close()
```

## HTTP Response — Writing

### ResponseWriter interface

```go
type ResponseWriter interface {
    Header() Header         // headers map
    Write([]byte) int       // write body (implements io.Writer)
    WriteHeader(statusCode) // send status code + headers
}
```

### Writing responses

```go
// Basic response
w.Header().Set("Content-Type", "application/json")
w.WriteHeader(http.StatusOK)
fmt.Fprintln(w, `{"status":"ok"}`)

// ⚠️ Write mà chưa WriteHeader → tự động gửi 200 OK
w.Write([]byte("data")) // implicit 200 OK

// Error responses
http.Error(w, "not found", http.StatusNotFound)
http.Error(w, "server error", http.StatusInternalServerError)

// Redirect
http.Redirect(w, r, "/new-url", http.StatusMovedPermanently)

// ✅ Redirect với context
http.Redirect(w, r, r.Context().Value("base_url").(string)+"/new", 302)

// JSON response (helper)
w.Header().Set("Content-Type", "application/json")
enc := json.NewEncoder(w)
enc.SetIndent("", "  ")
enc.Encode(response) // ⚠️ không set 200 — Write implicit
```

## Routing with ServeMux

### Basic routing

```go
mux := http.NewServeMux()

mux.HandleFunc("GET /", rootHandler)
mux.HandleFunc("GET /users/{id}", userHandler)  // ⚠️ Go 1.22+ patterns
mux.HandleFunc("POST /users", createUser)

mux.HandleFunc("GET /health", healthCheck)
mux.HandleFunc("GET /ready", readinessCheck)

http.ListenAndServe(":8080", mux)
```

### Path patterns (Go 1.22+)

```go
// Path patterns với wildcards
mux.HandleFunc("GET /users/{id}", func(w http.ResponseWriter, r *http.Request) {
    id := r.PathValue("id")   // Go 1.22+ ✅
    // hoặc mux.Vars(r)["id"]   // Go < 1.22
})

// Catch-all
mux.HandleFunc("GET /static/{*filepath}", serveStatic)

// Nested paths
mux.HandleFunc("GET /api/v1/users/{id}/orders/{order_id}", orderHandler)
```

### Sub-request routing (mounting)

```go
// Mount một mux vào một prefix
apiMux := http.NewServeMux()
apiMux.HandleFunc("GET /users", listUsers)
apiMux.HandleFunc("POST /users", createUser)

mux := http.NewServeMux()
mux.Handle("GET /api/", http.StripPrefix("/api", apiMux))
// /api/users → listUsers, /api/users/ → createUser
```

## Middleware Pattern

### Chain of Responsibility

```go
// Middleware = func(http.Handler) http.Handler
func loggingMiddleware(next http.Handler) http.Handler {
    return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        start := time.Now()
        next.ServeHTTP(w, r)
        log.Printf("%s %s %v", r.Method, r.URL.Path, time.Since(start))
    })
}

func authMiddleware(next http.Handler) http.Handler {
    return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        token := r.Header.Get("Authorization")
        if !validToken(token) {
            http.Error(w, "unauthorized", http.StatusUnauthorized)
            return
        }
        next.ServeHTTP(w, r)
    })
}

// Apply middleware
var handler http.Handler = http.HandlerFunc(apiHandler)
handler = loggingMiddleware(handler)
handler = authMiddleware(handler)

// Shorthand: chiều ngược apply (innermost → outermost)
handler := applyMiddleware(apiHandler,
    loggingMiddleware,
    authMiddleware,
)
```

### Middleware with typed context

```go
// Typed context keys — tránh string key collision
type contextKey string
const UserIDKey contextKey = "user_id"

func userIDMiddleware(next http.Handler) http.Handler {
    return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        userID := extractUserID(r) // từ token/cookie
        ctx := context.WithValue(r.Context(), UserIDKey, userID)
        next.ServeHTTP(w, r.WithContext(ctx))
    })
}

// Read from handler
func handler(w http.ResponseWriter, r *http.Request) {
    userID := r.Context().Value(UserIDKey).(string)
}
```

### ResponseWriter wrapper — capture status code

```go
type statusRecorder struct {
    http.ResponseWriter
    status int
}

func (r *statusRecorder) WriteHeader(code int) {
    r.status = code
    r.ResponseWriter.WriteHeader(code)
}

func statusMiddleware(next http.Handler) http.Handler {
    return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        rec := &statusRecorder{w, 200}
        next.ServeHTTP(rec, r)
        metrics.RecordStatus(r.URL.Path, rec.status)
    })
}
```

## HTTP Client

### Basic client

```go
resp, err := http.Get("https://api.example.com/users")
defer resp.Body.Close()

body, _ := io.ReadAll(resp.Body)
```

### Client with configuration

```go
client := &http.Client{
    Timeout: 30 * time.Second,         // total request timeout
    Transport: &http.Transport{
        MaxIdleConns:        100,
        MaxIdleConnsPerHost: 10,         // connection pool per host
        IdleConnTimeout:     90 * time.Second,
        TLSHandshakeTimeout: 10 * time.Second,
    },
}

resp, err := client.Get("https://api.example.com/users")
```

### http.RoundTripper — custom transport layer

```go
// RoundTripper = interface để customize cách request được gửi
// Default: http.DefaultTransport (HTTP/1.1, connection pooling)

// Custom RoundTripper: logging, retry, auth injection
type authTransport struct {
    base     http.RoundTripper
    token    string
}

func (t *authTransport) RoundTrip(req *http.Request) (*http.Response, error) {
    req = req.Clone(req.Context())
    req.Header.Set("Authorization", "Bearer "+t.token)
    return t.base.RoundTrip(req)
}

// Use
client := &http.Client{
    Transport: &authTransport{
        base:  http.DefaultTransport,
        token: "secret",
    },
}
```

### Request building

```go
// POST JSON
payload := Request{Name: "Alice", Age: 30}
data, _ := json.Marshal(payload)

req, _ := http.NewRequestWithContext(ctx, "POST", "https://api.example.com/users", bytes.NewReader(data))
req.Header.Set("Content-Type", "application/json")
req.Header.Set("Authorization", "Bearer "+token)

resp, err := client.Do(req)
defer resp.Body.Close()
```

### Middleware — client interceptor pattern

```go
// Retry transport
type retryTransport struct {
    base     http.RoundTripper
    maxRetry int
}

func (t *retryTransport) RoundTrip(req *http.Request) (*http.Response, error) {
    for i := 0; i <= t.maxRetry; i++ {
        resp, err := t.base.RoundTrip(req)
        if err == nil && resp.StatusCode < 500 {
            return resp, err
        }
        if i < t.maxRetry {
            time.Sleep(time.Duration(i+1) * 100 * time.Millisecond)
        }
    }
    return t.base.RoundTrip(req)
}
```

## httptest — Testing HTTP

### Testing handlers

```go
func TestUserHandler(t *testing.T) {
    handler := http.HandlerFunc(userHandler)

    req := httptest.NewRequest("GET", "/users/123", nil)
    rec := httptest.NewRecorder()

    handler.ServeHTTP(rec, req)

    if rec.Code != http.StatusOK {
        t.Errorf("expected 200, got %d", rec.Code)
    }

    var user User
    json.Unmarshal(rec.Body.Bytes(), &user)
    if user.ID != "123" {
        t.Errorf("expected user ID 123, got %s", user.ID)
    }
}
```

### Testing with context

```go
ctx := context.WithValue(context.Background(), UserIDKey, "user-42")
req := httptest.NewRequest("GET", "/profile", nil).WithContext(ctx)
rec := httptest.NewRecorder()
```

### TestServer — full integration testing

```go
ts := httptest.NewServer(http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
    w.WriteHeader(http.StatusOK)
    fmt.Fprintln(w, `{"status":"ok"}`)
}))
defer ts.Close()

resp, _ := http.Get(ts.URL) // ts.URL = "http://127.0.0.1:xxxxx"
defer resp.Body.Close()
```

## Common Patterns

### CORS middleware

```go
func corsMiddleware(next http.Handler) http.Handler {
    return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        w.Header().Set("Access-Control-Allow-Origin", "*")
        w.Header().Set("Access-Control-Allow-Methods", "GET, POST, PUT, DELETE, OPTIONS")
        w.Header().Set("Access-Control-Allow-Headers", "Content-Type, Authorization")

        if r.Method == "OPTIONS" {
            w.WriteHeader(http.StatusNoContent)
            return
        }
        next.ServeHTTP(w, r)
    })
}
```

### Rate limiting

```go
var limiter = rate.NewLimiter(100, 100) // 100 requests/sec

func rateLimitMiddleware(next http.Handler) http.Handler {
    return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        if !limiter.Allow() {
            http.Error(w, "rate limit exceeded", http.StatusTooManyRequests)
            return
        }
        next.ServeHTTP(w, r)
    })
}
```
